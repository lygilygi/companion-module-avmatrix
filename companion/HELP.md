AV Matrix Video Switcher

Supported devices
AV Matrix 0402U
AV Matrix 0403U

Connection
The module uses UDP.
Device UDP port: 19523
Local UDP port: 19522

Configuration
Device IP / Hostname
Enter the IP address (or hostname) of the switcher.

Sources
Inputs 1 to 4 are exposed as IN1 to IN4.
Still images are exposed as STILL1 to STILL16.

Feedback and polling
The module maintains state by receiving status packets from the device and by periodic polling.
Buttons created from presets include feedback styles.

Troubleshooting
If the status icon shows a warning, verify IP connectivity and that UDP ports are not blocked.
